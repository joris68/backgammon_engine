# Contributing

Thank you for your interest in contributing to this project.

## How to Contribute

1. **Open an Issue**  
   If you find a bug, have a question, or want to propose a new feature, please open an issue before submitting a pull request.

2. **Fork the Repository**  
   Create a personal fork of the repository and make your changes in a new branch.

3. **Follow the Code Style**  
   Keep your changes consistent with the existing codebase. Apply any relevant linters or formatters used by the project.

4. **Write Clear Commit Messages**  
   Use concise, descriptive commit messages that explain the purpose of each change.

5. **Add Tests if Applicable**  
   When fixing bugs or adding features, include tests to maintain code quality.

6. **Submit a Pull Request**  
   Open a pull request against the main branch. Describe your changes and reference any related issues.
